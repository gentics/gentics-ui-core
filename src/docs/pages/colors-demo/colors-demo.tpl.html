Colors

The UI Core defines the following colors, which can be overridden as desired:

States

primary / $gtx-color-primary
The primary color is the "brand" color, and should be the most widely-used color. It denotes the main elements and actions. For example, the main action button on a page should use the primary color.
secondary / $gtx-color-secondary
The secondary color is a neutral color used to denote secondary, less important elements and actions. For example, a "cancel" button may use the secondary color, whereas the "confirm" button would be primary.
success / $gtx-color-success
The success color is used to denote that an action has completed successfully; or in the case of a button, it may denote an affirmative outcome.
warning / $gtx-color-warning
The warning color is used to denote some outcome or action which may possibly be or result in an unexpected or unwelcome state. Whereas the alert color denotes a definite error or unwanted state, the warning color merely shows a possible error or unwanted state.
alert / $gtx-color-alert
The alert color is used to denote that an error has occurred, or that an undesirable state has been reached.
